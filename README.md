# PracticalCourseDatabaseImplementation

# Cmake compile instructions
```bash
mkdir buildRelease
cd buildRelease
cmake -DCMAKE_INSTALL_PREFIX=/usr/local -DCMAKE_BUILD_TYPE=Release ..
make
```

# Execute with
```bash
./practicalcoursedatabaseimplementation
```

or run a unit test with
```bash
./unitTest
```
